# CodeAlpha Task - 1. Iris Flower Classification

## Project Overview
This project predicts the species of Iris flowers using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Logistic Regression

## Results
The model achieved 100% accuracy on the test dataset.

## Features
- Data Loading
- Data Cleaning
- EDA
- Visualization
- Model Training
- Prediction
- Model Evaluation

## Results

The Logistic Regression model achieved high accuracy in classifying Iris flower species.

The model successfully classified:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

This project demonstrates the use of machine learning for multi-class classification problems.
